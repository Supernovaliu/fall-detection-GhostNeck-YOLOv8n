Before start using the script for training, please download the strathclyde fall detection dataset from Roboflow: 
https://universe.roboflow.com/university-of-strathclyde-m436x/fall_detection-shgkx .

Please download all raw images which are 474 images in YOLOv8 label format and change the file path as your own file path before training.

For cross dataset validation, please download Le2i labeled dataset from roboflow: https://universe.roboflow.com/new-workspace-qfcus/le2i
